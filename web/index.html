使用方法

  1. 选择源语言和目标语言
  2. 点击"开始识别源语言"按钮开始录音
  3. 说话，系统会自动识别并翻译
  4. 翻译结果会自动显示并可以试听语音

手动输入

您也可以在文本框中手动输入文本，然后点击翻译按钮。
